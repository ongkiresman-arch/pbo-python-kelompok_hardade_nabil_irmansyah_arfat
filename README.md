## DESKIRIPSI PROJEK
Portal Perpustakaan adalah project berbasis console yang dibuat menggunakan bahasa Python dengan menerapkan konsep Pemrograman Berorientasi Objek (PBO).

Project ini disusun dalam beberapa file Python (multi-file) untuk memisahkan logika program agar lebih rapi, terstruktur, dan mudah dikembangkan.
